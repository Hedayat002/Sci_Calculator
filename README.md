# 🧮 Python Scientific Calculator

A feature-rich, Windows-style Scientific Calculator built 
entirely with Python and Tkinter — no external GUI 
frameworks required.

## ✨ Features

### 🖩 Three Calculator Modes
- **Standard** — Basic arithmetic with memory functions
- **Scientific** — Advanced math, trigonometry, logarithms
- **Graphing** — Plot mathematical functions visually

### 🔄 13 Unit Converters
Currency | Length | Weight | Temperature | Volume | 
Area | Speed | Time | Power | Energy | 
Pressure | Angle | Data

### 🎨 UI/UX
- Windows 11 dark theme design
- Responsive layout (resizable window)
- Scrollable sidebar navigation
- Animated hover effects
- Calculation history panel
- Memory system (MC, MR, M+, M−, MS)
- Keyboard input support

## 🛠️ Modules Used

| Module | Purpose |
|--------|---------|
| `tkinter` | Main GUI framework |
| `tkinter.ttk` | Styled widgets (Combobox, Scrollbar) |
| `tkinter.messagebox` | Memory popup dialogs |
| `math` | sin, cos, log, sqrt, factorial, pi, e |
| `re` | Regex for preprocessing trig expressions |

## 🚀 How to Run

pip install python (3.8+)
python main.py
